2020/07/12 단어

By YEN 12 July 2020

  • 개인적

個人的 ↔ 공적 = 사적

  • 개최

舉辦 + 하다/되다

  • 객관적[객꽌적]

客觀的 ↔ 주관적

  • 거래

交易 + 하다/되다;끊다

  • 거품

泡沫 + 이 일다/생기다/빠지다

  • 건강식품

健康食品

  • 건망증[건망쯩]

健忘症 + 있다/없다;심하다

  • 건설

建設 = 건축,건물 ↔ 파괴 + 하다/되다

  • 건축

建築 = 건설,건물 ↔ 파괴 + 하다/되다

  • 걸음

腳步 + 멈추다/늦추다/빠르다

  • 검사

檢查 + 하다/되다

  • 검색

搜尋 + 하다/되다

  • 겁

害怕 + 이 나다/있다/없다 (升起害怕/有害怕->膽小/沒有害怕->膽大)

  • 겉모습[건모습]

外貌 = 외관/겉모양

  • 격려[경녀]

鼓勵 + 하다/받다/보내다

  • 결과

結果 + 가 나오다/좋다

  • 결론

結局、結論 + 이 나다;을 맺다 = 결말

  • 결말

結局 = 결말 + 이 나다

  • 결승[결씅]

決賽 + 에 나가다/오르다 ↔ 예선

  • 겸손

謙虛 + 하다 ↔ 거만

  • 경고

警告 + 하다/주다/받다

  • 경력[경녁]

經歷 + 이 있다/없다 = 경험/이력

  • 경비

經費 = 비용 + 가 들다;를 줄이다

  • 경영

經營 + 하다/되다

  • 경우

情況 = 상황 +이 있다/없다

  • 경제

經濟 + 가 종다/나쁘다;를 살리다

  • 경제적

經濟的、划算的 ↔ 비경제적

  • 경험

經驗 + 하다;이 풍부하다;을 쌓다

  • 계악

合約 + 하다;을 성사시키다;을 맺다

  • 계층

階層 *소외 계층 弱勢團體
